YouTube-Uploader
===============
easily upload videos to youtube without the need of the official YouTube API.


Installing 
---------------

1. python -m venv env
2. source env/bin/activate
3. python -m pip install -r requirements.txt

How to run
---------------

python3 main.py -u mail@gmail.com:password -f video.mp4
